**Учет студентов группы**
.О проекте
Веб-приложение и консольная программа на C++ для учета студентов в группе. ПРОЕКТ по предмету "Обеспечение качества функционирования компьютерных систем".

Структура проекта
папка website(html - код, css - стили, JS) 01 - тех задание 02 - код с комментарием


Действие	Инструкция
Добавить ученика	Введите имя в поле "Имя и фамилия ученика" → нажмите "➕ Добавить ученика".
Выставить оценку	Нажмите кнопку "➕ оценка" в строке ученика → введите число от 1 до 5.
Быстрая оценка	Кликните на строку ученика (она подсветится) → выберите предмет, введите оценку → нажмите "Поставить выбранному".
Удалить ученика	Нажмите 🗑️ в строке ученика и подтвердите.
Посмотреть статистику	Карточки вверху страницы обновляются автоматически.
Попробовать демо	Нажмите "📋 Загрузить демо-класс".
